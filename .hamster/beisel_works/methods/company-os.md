# Company OS - Toolsets used in our operations



This method describes on a high level the decided technology stack that we try to align ourselves to. It is not conclusive - more can be added per project. 

It is mostly prescriptive, Using alternatives than those decided upon, needs reasoning, justification and Founder Sign-Off.

## Base OS in development tasks:

- TypeScript
- Node.js 22 LTS
- Next.js App Router + React
- Tailwind CSS v4
- pnpm
- Vercel
- Supabase
- Stripe
- [Trigger.dev](http://Trigger.dev)
- Vitest + Playwright
- ESLint + Prettier
- OpenTelemetry + Sentry
